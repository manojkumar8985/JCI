# JCI Directory 🚀

A premium business directory for JCI members to discover services, manage profiles, and connect with other professionals.

## Features
- **Dynamic Pricing**: Custom service pricing for business profiles.
- **Member Discounts**: 10% discount for JCI members.
- **Admin Dashboard**: Comprehensive management for admins to approve and monitor businesses.
- **Real-time Notifications**: Service request tracking and status updates.

## Tech Stack
- React
- Firebase (Auth, Firestore)
- Tailwind CSS
- Lucide React
